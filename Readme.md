## JoJpeg Addons
This is a collection of usefull little addons for blender, usually its single file addons.
Most of the code is AI generated but I use those addons in my daily life and they should work fine. 



#### [HoldHide](blender-h-addon/Readme.md) 
 Prevent accidentally hiding objects by fat-fingering when trying to hit "G". Instead you need to hold H for a little while to make to actually Hide the object. A little visual indicator will pop up to let you know. Currently it is Red because i want to train myself to not hit H, but you can set color and size in the preferences.

#### [Variable Fields](blender-variable-fields/Readme.md) 
 Allows you to use variables for data paths, making it easier to create variants of objects and manage different versions of your assets. Also good for working with a team that is not fluid in blender but wants to be able to change some values without having to understand the whole prorgam.


#### [Preview World to Render](preview_world_to_render/Readme.md) 
 Copy the currently active world in the material-preview to your world so that you can use it in rendering.

#### [Focus Similar F-Curves](focus_similar_fcurves/Readme.md) 
 Select all f-curves that are similar to the currently active f-curve. eg all curves named "X-Location". Also if you have more than one curve selected.
